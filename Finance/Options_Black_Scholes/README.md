## Проект: Оценка стоимости опциона и расчет греков используя информацию с API MOEX

### Описание проекта

Проект представляет из себя набор функций для получения информации с Московской биржи посредством API стоимости опционов put и call, расчета греков данных опционов

### Теория: Что такое модель Блэка-Шоулза и греки

**Модель Блэка-Шоулза** — это математическая модель, используемая для оценки стоимости финансовых опционов. Она предполагает, что цены активов следуют геометрическому броуновскому движению, а рынок является эффективным. Модель позволяет определить теоретическую цену опциона, учитывая такие параметры, как текущая цена актива, страйк, волатильность, безрисковая ставка и время до истечения.

**Греки** — это чувствительности цены опциона к различным параметрам:  
- **Delta**: Чувствительность цены опциона к изменению цены базового актива.  
- **Gamma**: Скорость изменения Delta по отношению к цене базового актива.  
- **Theta**: Изменение цены опциона с течением времени (временной распад).  
- **Vega**: Чувствительность цены опциона к изменению волатильности.  
- **Rho**: Чувствительность цены опциона к изменению безрисковой ставки.
